Mas afinal, o que é API TECLA ZAP?

Para explicar de forma simples

O API TECLA ZAP é um serviço que facilita a interação com o WhatsApp através de uma API fácil de usar. Ele também fornece notificações instantâneas sobre atividades relacionadas ao seu número.

É importante lembrar que o API TECLA ZAP não deve ser usado para enviar spam ou mensagens indesejadas, nem para violar os termos de serviço do WhatsApp.

Use a API com responsabilidade, criando recursos que sejam úteis tanto para seus clientes quanto para os usuários do WhatsApp.

abaixo:

Quem pode utilizar API TECLA ZAP?

Não há restrições quanto ao uso da nossa API, porém, tipicamente, identificamos dois grupos distintos de usuários:

  • Programadores com familiaridade em API's RestFul. Se você não se encaixa nessa categoria, mas conhece alguém com essas habilidades, também é bem-vindo :)
  • Utilizadores de soluções de terceiros que possibilitam a integração com a API TECLA ZAP.

Quais sãos os limites?

Quero esclarecer que não há limite para o número de mensagens enviadas!

  • No entanto, é crucial compreender que você está utilizando uma sessão do WhatsApp Web, então o padrão de uso precisa ser compatível. Além disso, sempre recomendamos a leitura cuidadosa das políticas estabelecidas pelo WhatsApp em sua página oficial: https://www.whatsapp.com/legal

Como funciona o fluxo de envio?

Envio de Mensagem

Você envia via API uma mensagem para o API TECLA ZAP.

Adição à Fila

O API TECLA ZAP adiciona em uma fila e te retorna o ID da mensagem.

Processamento

Sua instância processa a fila enviando para o WhatsApp.

Notificação de Delivery

Seu Webhook de delivery é chamado quando a mensagem é processada, te avisando que foi enviada ou que houve falha.
